# AKAN NAME GENERATOR
## Author
**Ryan Amani**
## Description
The Akan Name Generator is a simple web application that allows users to discover their Akan name based on their date of birth and gender. Akan names originate from Ghanaian culture and are traditionally assigned according to the day of the week a person is born. This project aims to celebrate African heritage while demonstrating basic HTML, CSS, and JavaScript functionality.
